# boom-challenge-submission

Material Fragment Predictor is a machine-learning product built for the Boom Challenge that estimates how a material will fracture under different blasting conditions. By taking scenario inputs—such as porosity, atmosphere, gravity, coupling, strength, shape factor, energy, and angle—the model predicts core fragmentation statistics including P80, fines fraction, oversize fraction, and R95 / R50 (separately for fines and oversize).

The predictor works by transforming raw inputs into a standardized feature space, learning the nonlinear relationship between conditions and fragment outcomes, and producing calibrated predictions for new scenarios in seconds. Instead of relying on expensive trial-and-error or repeated experimentation, users can evaluate configurations quickly and make more informed decisions.
